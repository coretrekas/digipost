### Added

- Initial release of the Digipost PHP SDK
- Support for sending messages to Digipost recipients
- Support for identifying recipients by personal ID, Digipost address, email, or phone
- Support for document attachments
- Support for print fallback when recipient doesn't have Digipost
- Support for SMS and email notifications
- Support for structured data types (Invoice, Appointment, etc.)
- Support for batch sending
- Support for archive operations
- Support for inbox operations
- Support for document events tracking
- Support for recipient autocomplete
- PKCS#12 and PEM certificate authentication
- PHPStan level 9 static analysis
- Pest PHP test suite
- Laravel Pint code style
